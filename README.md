# Student Performance Analysis using Data Science

## Project Overview

This project is completed as part of the Horizon TechX Artificial Intelligence & Data Science Internship Program.

The project focuses on Data Analysis and Exploratory Data Analysis using a student performance dataset. It includes data cleaning, preprocessing, statistical analysis, visualization, insight generation, grade classification, performance categorization, and analytical reporting.

## Objective

The main objective of this project is to analyze student academic performance and identify useful trends, patterns, and insights from the dataset.

## Key Features

- Data loading using Pandas
- Missing value handling
- Duplicate record removal
- Average score calculation
- Pass/Fail result generation
- Grade classification
- Performance category generation
- Statistical analysis
- Data visualization
- Correlation analysis
- Top 5 student analysis
- Professional analytical report generation

## Tools and Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- VS Code
- GitHub

## Dataset Description

The dataset contains student academic performance data. It includes student-related factors such as gender, parental level of education, lunch type, test preparation course, math score, reading score, and writing score.

## Project Workflow

1. Import required Python libraries
2. Load the student performance dataset
3. Check missing values and duplicate records
4. Clean and preprocess the dataset
5. Generate average score, result, grade, and performance category
6. Perform statistical analysis
7. Create visualizations
8. Identify insights and trends
9. Generate analytical reports
10. Save cleaned dataset and output files

## Visualizations Created

- Gender Distribution

- Grade Distribution
- Test Preparation Course vs Average Score
- Lunch Type vs Average Score
- Parental Education vs Average Score
- Subject-wise Average Score
- Math Score vs Reading Score
- Correlation Heatmap

## Output Files

The project generates the following output files:

- Cleaned student performance dataset
- Professional analysis report
- Top 5 students report
- Performance summary report
- Multiple visualization charts

## Key Insights

- Students who completed the test preparation course generally scored better.
- Reading and writing scores show a strong relationship.
- Subject-wise analysis helps compare math, reading, and writing performance.
- Grade distribution gives a better understanding of academic levels.
- Performance categorization helps identify high, average, and low performers.
- Data visualization helps in understanding trends clearly.

## Conclusion

This project successfully demonstrates the complete workflow of Data Analysis and Exploratory Data Analysis. It includes data preprocessing, statistical analysis, visualization, insight generation, and reporting using Python.

The project can be useful for educational institutions to understand student performance patterns and make data-driven academic decisions.
